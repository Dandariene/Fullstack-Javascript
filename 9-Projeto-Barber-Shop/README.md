# Projeto Barber Shop
## Tecnologias utilizadas no projeto: HTML e CSS


Este foi um projeto bastante desafiador pois foi utilizado bastante o  ***flexbox*** , algo que no início tive dificuldade, mas ao final do projeto já me sinto mais confortável usando a propriedade. Foi muito válida a experiência.<br>
*Agregou muito ao meu aprendizado.*<br>
